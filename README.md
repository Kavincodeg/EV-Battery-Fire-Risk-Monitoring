# 🔋 EV Battery Degradation & Fire Risk Monitoring System

A software-based IoT and Machine Learning project to monitor EV battery health,
predict degradation, and detect potential fire risks.

## 🚀 Features
- Battery degradation prediction (ML)
- Fire risk classification (Safe / Warning / Critical)
- Real-time Streamlit dashboard
- Software-based IoT simulation
- Buzzer alert simulation

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy

## 📊 Dataset
Trained using a real experimental battery dataset.
Real-time values are simulated for monitoring.

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run dashboard.py
